# Go (Golang)

## What is Go?

Go (also called Golang) is a statically typed, compiled programming language designed for building efficient, reliable, and scalable software. It was developed at Google in 2007 by Robert Griesemer, Rob Pike, and Ken Thompson and officially released in 2009.

Go was created to simplify system programming while maintaining high performance similar to languages like C and C++. It is widely used for backend development, cloud services, networking tools, and distributed systems.
